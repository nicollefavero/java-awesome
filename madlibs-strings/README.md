# Madlibs

This is a Java program that explores more about read files, string concatenation and programming logic.

## Build
```shell
gradle build
```

## Run
```shell
cd build/libs
java -jar madlibs.jar
```

## How to Use
Just write anything when the program asks for a word and have fun!
